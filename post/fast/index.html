FAST-First Arrival Seismic Tomography Install in Linux

First Arrival Seismic Tomography FAST是一个能夠計算二維/三維first arrival tomography的程式，他已經20歲了(1998年first release)，請下載FAST主程式，也必須下載for g77的patch檔。

1. 準備好編譯器以及所需的libraries

CentOS7兼容包或者直接下載

CentOS 6.x

1
2
3

yum install compat-libf2c-34 compat-gcc-34-g77-3.4.6-20.fc14.i686 make -y
ln -s /usr/bin/g77 /usr/bin/f77
ln -s /usr/lib/libg2c.so.0 /usr/lib/libf2c.so.2

2. 下載FAST以及其patch

1
2

wget http://terra.rice.edu/department/faculty/zelt/fast.tar.gz
wget http://terra.rice.edu/department/faculty/zelt/fast_g77_patch.tar.gz

解壓縮

1
2
3

tar zxvf fast.tar.gz
mv fast fast_g77
tar zxvf fast_g77_patch.tar.gz

3. 編譯

1
2
3
4

cp -Rf fast_g77/* fast
patch -p0 < fast_g77.diff
cd fast
./COMPILE_FAST_Lee.sh
